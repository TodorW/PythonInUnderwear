app.py 
from piu import PIU, Request, Response
from piu import SessionMiddleware, require_auth

app = PIU()
app.middleware.use(SessionMiddleware(secret_key="🩲"))

@app.get("/")
def index(request: Request):
    return Response.json({"hello": "world"})

@app.get("/user/<id>")
@require_auth
async def get_user(request: Request, id: str):
    return Response.json({"id": id, "role": request.session["role"]})

app.run() # http://127.0.0.1:5000

Features

⚡
Sync & Async
Mix sync and async handlers freely. Full WSGI and ASGI support out of the box.
🛣️
Smart Routing
Decorator-based URL rules with dynamic path params and Blueprint grouping.
🔐
Auth & Sessions
HMAC-signed cookie sessions, CSRF protection, and @require_auth built in.
🚦
Rate Limiting
Global and per-route limits with sliding window. Redis-swappable store.
🧪
Test Client
In-process HTTP client with cookie jar. No server needed. pytest-ready.
🔄
Hot Reload
Watches .py files and restarts on change. Enabled by DEBUG=true in .env.
